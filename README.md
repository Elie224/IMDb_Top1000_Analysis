# Analyse des 1000 meilleurs films IMDb (2020)

Ce projet présente une analyse exploratoire (EDA) du dataset `imdb_top_1000.csv` qui recense les 1000 meilleurs films selon IMDb (données de 2020).

## Objectifs
- Comprendre la structure du dataset
- Identifier les réalisateurs les plus représentés
- Étudier la variable 'Genre' et la rendre exploitable
- Nettoyer la variable 'Runtime' et créer des catégories de durée
- Créer des catégories de notation et analyser la distribution
- Étudier les relations entre les variables qualitatives

## Technologies utilisées
- Python 3
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## Contenu
- `data/imdb_top_1000.csv` : le dataset
- `notebooks/imdb_eda.ipynb` : notebook avec l'analyse
